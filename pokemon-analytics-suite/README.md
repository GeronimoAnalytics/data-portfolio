# The Ultimate Pokemon Analytics Suite

Welcome to **The Ultimate Pokemon Analytics Suite**! This is a custom, high-fidelity Power BI data visualization dashboard designed to break down Pokemon stats, attributes, and team combinations with a premium, arcade-style UI.

Whether you are min-maxing your competitive team or just want to see how Pikachu's speed stacks up against your favorite Pokemon, this suite turns raw pocket monster data into actionable battle intelligence.

## Features

- **Poke Details Screen**: Deep-dive into individual Pokemon metrics, base stat totals, and Body Mass Index (BMI) tracking.
- **Dynamic Stat Bars**: Visualized attribute breakdowns covering HP, Attack, Defense, Special Attack, Special Defense, and Speed.
- **Advanced Analytics Overlays**: Custom UI components including Head-to-Head comparisons, BMI vs. Speed scatter insights, Element Distribution matrices, and Team Selection optimization.
- **Sleek Retro-Modern UI**: Custom-themed dashboard with animated accents, custom icon wrappers, and a fully interactive sidebar selector for seamless navigation through the Pokedex.

## Built With

- **Power BI Desktop** - Core data modeling and dashboard orchestration.
- **DAX (Data Analysis Expressions)** - Advanced data calculations, dynamic filtering, and custom KPI metrics.
- **HTML/CSS & SVG Canvas** - Custom visuals, clean curved borders, and tailored UI element grouping.
- **The Pokedex API / Custom Dataset** - Cleaned, optimized, and structured Pokemon attribute data.

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/GeronimoAnalytics/data-portfolio.git
```

2. Open this project folder:

```bash
cd data-portfolio/pokemon-analytics-suite
```

3. Open the report in Power BI Desktop:

- `pokemon.pbix`

4. Explore the data:

- Use the left-hand sidebar list to slice and dice stats by specific Pokemon.
- Toggle the bottom tabs to change your analytical perspective.

## Insights and Goals

This project was built to showcase how standard business intelligence tools (like Power BI) can be pushed past their default corporate templates into highly immersive, themed user experiences. By leveraging custom canvas designs, HTML inserts, and complex data relationships, it demonstrates that data visualization can be both incredibly informative and visually striking.
